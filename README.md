# 💤 Análise do Sono e Saúde

Este projeto realiza uma análise exploratória de dados sobre sono e saúde utilizando um conjunto de dados contendo informações como duração do sono, qualidade do sono, nível de atividade física e indicadores de saúde.

O objetivo é compreender relações entre hábitos de sono e aspectos da saúde física e mental por meio de visualizações e estatísticas descritivas.

---

## 📊 Dataset

O dataset utilizado é o arquivo: sleepHealth.csv
Ele contém informações relacionadas a:
- Duração do sono
- Qualidade do sono
- Nível de atividade física
- Estresse
- IMC
- Batimentos cardíacos
- Ocorrência de distúrbios do sono
- Perfil demográfico dos indivíduos

---

## 🎯 Objetivo

- Explorar os dados de sono e saúde
- Identificar padrões e correlações
- Visualizar distribuições e relações entre variáveis
- Obter insights sobre como hábitos de sono impactam a saúde

---
